  Design notes:

  - Parser — segment-based, handles ID1 (serial), ID5 (DEX date+time
  YYYYMMDD*HHMM), VA1 (paid vends), PA1/PA2 pairs → lanes.
  - Repository — opens connection, begins transaction, calls dbo.SaveDEXMeter →
  gets @@IDENTITY via OUTPUT INSERTED.Id, then calls dbo.SaveDEXLaneMeters with
  TVP dbo.DEXLaneMeterType. Catches SQL errors 2627/2601 → returns Conflict.
  - Auth — custom AuthenticationHandler<BasicAuthSchemeOptions>, fixed-time
  compare, challenge emits WWW-Authenticate: Basic.
  - Endpoint — POST /vdi-dex?machine=A, body = DEX text. Returns 201 Created,
  400, 409, 401.
  - Transaction — parent + lanes atomic per spec.
